Projet ICGO

Présentation de l’Institut I-CGO

L’institut Claude Gaston Octave (I-CGO) est un centre de formation basé à Dijon. Il propose des stages en informatique pour entreprises et particuliers. L’institut est dirigé par ses trois fondateurs : Claude, Gaston et Octave. Dans une volonté d’expansion, deux nouvelles agences ont été ouvertes à Beaune et à Auxerre. Dijon reste le siège principal.

Contexte du projet

Le projet « ProjetICGO » est une application de bureau développée en C#. À mon arrivée, l'application n'était pas encore fonctionnelle. Ma mission principale à été de réaliser la maintenance de cette application, en commençant par analyser l’existant.

Travail réalisé

  • Analyse de l'existant et compréhension du code source
  • Documentation du modèle de données actuel (non documenté auparavant)
  • Installation de l'application et de la base de données MySQL en local
  • Mise en place d’un gestionnaire de version pour le travail collaboratif
  • Réalisation de tests fonctionnels pour identifier les parties opérationnelles

Compétences développées

  • Compréhension et analyse d’une application existante non documentée
  • Utilisation de C# et Visual Studio pour des projets réels
  • Conception et interprétation de modèles relationnels
  • Utilisation d’un gestionnaire de versions (Git)
  • Installation et configuration d’un serveur MySQL local
  • Rédaction de documentation technique complète
  • Travail collaboratif sur un projet en maintenance évolutive
