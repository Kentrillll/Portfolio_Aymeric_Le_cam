Travail sur les APIs

Qu'est-ce qu'une API ?

Une API (Application Programming Interface) est une interface permettant à différentes applications de communiquer entre elles. Elle définit un ensemble de règles et de fonctions que les développeurs peuvent utiliser pour accéder à certaines fonctionnalités ou données, souvent à travers le web.

Exercices réalisés

  • API sur les vélos en libre service : Nous avons utilisé une base de données publique pour récupérer la disponibilité des vélos dans différentes villes. Cela nous a permis de manipuler les requêtes HTTP et de traiter les réponses JSON.
  • API musicale personnalisée : J'ai conçu une petite API qui permet de rechercher des musiques sur Deezer selon leur genre musical. Ce projet m’a permis de mieux comprendre les appels à des API externes et la structuration de réponses dynamiques.
  • Application de contact (connexion utilisateur) : La dernière API développée permettait la création d’une connexion sécurisée. Elle gère l’authentification et la vérification des identifiants, un début vers la sécurisation des services web.

Compétences développées

  • Compréhension et manipulation du protocole HTTP (GET, POST)
  • Utilisation et conception d’API REST
  • Gestion des formats JSON pour l’échange de données
  • Utilisation d’API externes (ex : Deezer)
  • Création d’API sécurisées avec gestion d’utilisateurs
  • Débogage et documentation des routes
